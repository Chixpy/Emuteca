Emuteca's source code documentation

Here is the documentation of the Emuteca's source code.

PasDoc was selected for documentation by two reasons:

  1. I don't like Lazarus' LazDoc with the comments in other file.

  2. Lazarus itself recognizes PasDoc comments, specially {< to show them in tooltips (I like the comments after the declarations, but before the implementation). Then, when the mouse pointer is over something the tooltips are not messed with the previous thing declared.

Previously, when Delphi was used for compilation, JADD has been used. Maybe it's more powerfull as you can more things that to document the code; for example, to do the user's manual or the offline help...
