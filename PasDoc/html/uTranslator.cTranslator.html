type cTranslator = class(TObject)

Description

Entity that translates texts and component captions.

The property Section must be set before calling Translate methods.

function Translate(const Key, sDefault: UTF8String): UTF8String; overload;

Translates a string.

Searches the Key, in the current Section, for a translation of the text.

If the Key is not found, then it will be created in the opened .ini file with the value of sDefault.

Parameters

Key where translation is stored.
