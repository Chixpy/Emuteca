System base folder

Folder where all data and subfolders reside. Used by default to store some data if the file isn't defined (System image or text.)

property DataFile: String read FDataFile write SetDataFile;

Name of the files with the game/group data.

This is the name of the files of games and groups data. No full path, no extension; only name. These files are handled by cGameManager, not cSystem.
