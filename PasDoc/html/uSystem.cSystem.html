property BaseFolder: UTF8String read FBaseFolder write SetBaseFolder;

System base folder

Folder where all data and subfolders reside. Used by default to store some data if the file isn't defined (System image or text.)

property Company: UTF8String read FCompany write SetCompany;

Company of the system.

property DataFile: UTF8String read FDataFile write SetDataFile;

Name of the files with the game/group data.

This is the name of the files of games and groups data. No full path, no extension; only name. These files are handled by cGameManager, not cSystem.

property DemoMusicFolder: UTF8String read FDemoMusicFolder write SetDemoMusicFolder;

Folder with demo music of the games.

This folder is inteded to store the music that will be played when selecting a game or group.
