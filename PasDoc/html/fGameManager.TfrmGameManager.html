function Group(const aIndex: integer): cGameGroup;

Gets a Group by index.

Usually used for iterations. Remember, it begins from 0 to [GroupCount - 1].

NOTE:

In TfrmGameManager, a group refers to the groups used for listing the games. Don't confuse with cGameManager.GroupAtPos.

Parameters
aIndex
The position on the list of groups.
Returns

The cGameGroup in this position. Nil if it don't exists, in other words, aIndex is out of range.
