Unit

uaEmutecaCustomGroup

Declaration

type caEmutecaCustomGroup = class(caCHXStorableTxt)

Description

This class defines an abstract basic group with all basic properties, but without a software list to avoid circular reference with ucEmutecaSoftware, using this in cEmutecaSoftware.CachedGroup property.
