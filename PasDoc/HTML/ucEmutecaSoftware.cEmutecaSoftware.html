ucEmutecaSoftware

Declaration

type cEmutecaSoftware = class(caEmutecaCustomSoft, IFPObserver)

Description

This class defines a parented sofware with links to its System and Group. In CachedSystem and CachedGroup properties, and autoupdating the reference keys from both with Observer Pattern.

Overrides the GetTitle method to get actual title from group if Title property is empty.

It can check if the filenames for media match with groups one.
